Internal gestionale (management) app for a web agency called PiraWeb.

**Tech Stack:** Next.js (App Router), TypeScript, Tailwind CSS v4, Supabase (auth + DB + storage), Recharts for charts, Lucide icons.

**Brand Identity:**
- Background: #000000 (pure black)
- Surface layers: #141414, #1a1a1a, #222222
- Border: rgba(240,237,230,0.1)
- Primary text: #f0ede6 (warm white)
- Muted text: rgba(240,237,230,0.5)
- Dim text: rgba(240,237,230,0.3)
- Accent: #c8f55a (lime green) — primary CTA color
- Accent hover: #d4ff6a
- Purple: #8c7af5
- Gold: #d4af37
- Font headings: Syne
- Font display: Bebas Neue
- Font body: Inter

**App sections:** Dashboard, Clients, Projects, Tasks, AI Scripts, Bacheca, Presenze (attendance), Analytics (Efficienza), Cashflow, Settings

**Role system:** admin, social_media_manager, content_creator, graphic_social, graphic_brand
- Admins see everything including financial data
- Employees have limited views

**Key findings from full audit (2026-04-03):** See ui_ux_audit_findings.md

**Why:** This is a production internal tool for the PiraWeb team.
**How to apply:** When suggesting design changes, respect the dark-first monochrome palette with lime accent. Don't suggest light mode as primary. Keep Italian language labels as-is.
